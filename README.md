# Questionnaire_System

動態問卷

網站基本功能

1.前台可搜尋問卷、作答以及觀看問卷內各個問題的統計數據

2.後台可設計、編輯和刪除問卷以及觀看問卷內各個問題的統計數據

3.後台可以新增、編輯和刪除常用問題

4.後台可以觀看問卷的作答內容，並匯出資料
